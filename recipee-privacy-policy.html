プライバシーポリシー

Recipee（レシピー）  |  最終更新日: 2026年6月10日

本プライバシーポリシーは、Recipee（以下「本アプリ」）における個人情報および利用者データの取り扱いについて説明します。 本アプリをご利用いただくことで、本ポリシーの内容にご同意いただいたものとみなします。

1. 収集する情報

本アプリが収集・利用する情報は以下の通りです。

情報の種類 内容 保存先
食材テキスト レシピ提案のために入力された食材リストおよび希望・条件 外部 API（一時処理のみ）
食材写真 写真入力モードで選択または撮影された画像 外部 API（一時処理のみ）
保存レシピ ユーザーがお気に入りに保存したレシピ情報 端末内のみ
検索履歴 過去の検索内容と提案されたレシピ（最大50件） 端末内のみ
プラン情報 フリー／プレミアムプランの状態、当日の利用回数 端末内のみ

本アプリは氏名・メールアドレス・位置情報・連絡先などの個人を特定できる情報を収集しません。

2. 情報の利用目的

  • 入力された食材・写真をもとに AI によるレシピ提案を行うため
  • 保存レシピおよび検索履歴を端末内に保持し、アプリの機能を提供するため
  • フリープランの1日あたりの利用回数を管理するため

3. 第三者への情報提供

本アプリはレシピ提案のために、入力された食材テキストおよび食材写真（base64 形式）を Anthropic, Inc.（米国）が提供する Claude API に送信します。 送信データは API によるレスポンス生成にのみ使用され、Anthropic 社のサービス利用規約およびプライバシーポリシーが適用されます。

上記以外の第三者に対して、利用者の情報を販売・提供・開示することはありません。 ただし、法令に基づく要請があった場合を除きます。

  • Anthropic プライバシーポリシー: anthropic.com/privacy

4. データの保存と管理

保存レシピ・検索履歴・プラン情報はすべて利用者の端末内（AsyncStorage）にのみ保存されます。 開発者のサーバーへの送信・保存は一切行いません。 アプリを削除すると、端末内に保存されたすべてのデータは消去されます。

5. カメラおよびフォトライブラリへのアクセス

写真入力機能を使用する場合、本アプリはカメラおよびフォトライブラリへのアクセス許可を要求します。 取得した画像はレシピ提案の目的にのみ使用され、端末外への保存は行いません。 アクセス許可は端末の設定からいつでも変更できます。

6. お子様のプライバシー

本アプリは13歳未満の方を対象としていません。 13歳未満の方からの情報を意図的に収集することはありません。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて更新されることがあります。 重要な変更がある場合は、本ページの「最終更新日」を変更することでお知らせします。 定期的にご確認いただくことをお勧めします。
